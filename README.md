# AI Problem Solving Assignment

## Problem 1: Tic Tac Toe AI

### Description
This project implements a Tic Tac Toe game where the user plays against an AI.

### Algorithms Used
- Minimax (planned)
- Alpha-Beta Pruning (planned)

### Features
- Interactive web interface
- AI opponent
- User vs Computer gameplay

### How to Run
1. Install Flask:
   pip install flask

2. Run:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/

### Output
User can play Tic Tac Toe against AI.
## Algorithm Comparison

| Feature | Minimax | Alpha-Beta |
|--------|--------|-----------|
| Execution Time | Higher | Lower |
| Nodes Explored | More | Less |
| Efficiency | Slower | Faster |

## Conclusion
Alpha-Beta pruning improves performance by reducing unnecessary node exploration while giving the same optimal result as Minimax.
